## Descrição do Projeto
Aplicativo desenvolvido em Ionic/Angular para controle de atendimento em laboratórios médicos, seguindo as regras de prioridade (SP, SG, SE) e formato de senha YYMMDD-PPSQ.

## Tecnologias
- Ionic Framework (v7+)
- Angular
- Capacitor


## Como rodar
1. Instale as dependências: `npm install`
2. Rode o projeto: `ionic serve`
